Limechat, IRC Client for Mac OS X

Jan 22 2013
By Erwann Mest
  • irc

Nope, IRC ain’t dead. Nope, Twitter is not enough. Yes, I’m still and I will still be on IRC. IRC isn’t a fashion stuff, IRC is a great protocol to communicate with other people (well at least, geeks). You can receive all supports you need about softwares, libraries, applications, frameworks. It’s not dead and won’t be because it’s still the best way to group lots of people in the same place. Doing that on Twitter is like spamming everyone. So, yes, please, come on IRC, mostly freenode. You won’t regret.

Anyway, as any activities on Internet, you must have the best tool to do it. About IRC, it’s probably LimeChat which answers the best to this question.

LimeChat is an open source IRC client for Mac OS X which gives you everything you need. Plus I know you love themes, themes are available and done in CSS. Yep.

Bonus for Windows fella

On Windows, I advise you HexChat which is a fork of XChat but free. (The Windows build of XChat is not free).
